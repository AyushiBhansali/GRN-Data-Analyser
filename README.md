# Procurement Intelligence & GRN Analytics Engine

A Python-based analytics tool that transforms raw ERP/GRN exports into structured procurement insights and automated Excel reports.

## Features

* Runner / Repeater / Stranger analysis
* Monthly buying trend analysis
* Supplier fragmentation tracking
* Supplier dependency analysis
* Pricing anomaly detection
* Automated multi-sheet Excel export
* ERP/SAP data cleaning and preprocessing

## Tech Stack

* Python
* Pandas
* NumPy
* OpenPyXL
* Google Colab / VS Code

## Workflow

1. Upload ERP/GRN Excel file
2. Clean and standardize procurement data
3. Generate procurement analytics
4. Export structured business insights to Excel

## Output Reports

* Part Classification
* Supplier Fragmentation
* Supplier Dependency Risk
* Monthly Procurement Trends
* Pricing Anomalies
* Cleaned Procurement Data
